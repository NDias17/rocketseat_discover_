Sintaxe pra criação de funções em JS:
function nomeDaFuncao (){
	//blocoDeCodigo
}
Para executar a função:
nomeDaFuncao()
--------------------------------------------------------------
function nomeDaFuncao(valores){
	return [operação]
}
Para executar:
nomeDaFuncao(valores)

Exemplo:
function soma(a, b, c){
	return a + b + c
}
soma(1, 2, 3)
